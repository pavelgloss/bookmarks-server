
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Database
* Using NeDB from https://www.npmjs.com/package/@seald-io/nedb for in-memory document storage (with same api as mongoDB)

## Swagger
* localhost:3000/api



## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
